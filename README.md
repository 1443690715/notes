## 目录结构

工程中文件目录的安排如下：

	README.md     Markdown文件索引目录

	README.html   html文件索引目录

	html/         所有笔记的html文本格式

	essays/       所有笔记的Markdown文本格式

	codes/        存放可供下载的代码

	images/       存放文档中的所有图片
 
	enclosure/    存放可供下载的附件（pdf等）
